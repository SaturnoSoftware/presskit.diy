## Tips for Writing Good Descriptions

### Company Description
- Focus on your mission and values
- Highlight what makes you unique
- Include concrete achievements
- Keep it conversational and authentic

### Product Description
- Start with a hook (what makes it special?)
- Explain the core gameplay/concept
- List key features prominently
- Include system requirements if relevant
- End with a call-to-action

### Features
- Be specific, not generic
- Lead with benefits, not just features
- Use active, engaging language
- Keep each feature to 1-2 sentences
